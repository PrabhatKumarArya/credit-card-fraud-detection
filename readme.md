# Credit Card Fraud Detection System

## 🔍 Project Overview
ML model to detect fraudulent transactions using classification algorithms.

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit / Flask

## 📊 Model
- Random Forest / XGBoost + SMOTE
- Accuracy: 87%
- Precision/Recall optimized for fraud detection

pip install -r requirements.txt
